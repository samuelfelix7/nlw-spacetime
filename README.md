## 🖥️ Projeto
Esse é um projeto WEB responsivo de uma capsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rockeseat com as seguintes tecnologias: 

- HTML 
- CSS
- Git e Github
